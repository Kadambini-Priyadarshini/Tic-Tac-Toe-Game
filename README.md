Tic Tac Toe Game ❌⭕

A simple C++ console-based Tic Tac Toe game for two players. Players enter their names and take turns placing their marks (X and O) on a 3×3 board. The game checks for a winner after each move and declares the winner or a draw when the game ends.

Features

- Two-player gameplay
- Player name input
- 3×3 game board
- Alternating turns between players
- Win and draw detection
- Simple console-based interface

Technologies Used

- C++
- VS Code

How to Run

1. Compile the program:

g++ ttt.cpp -o ttt

2. Run the executable:

./ttt

Game Rules

- The game is played by two players.
- Player 1 uses X and Player 2 uses O.
- Players take turns selecting an empty position on the board.
- The first player to get three marks in a row, column, or diagonal wins.
- If all positions are filled without a winner, the game ends in a draw.
