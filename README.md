# Rock Paper Scissors Game ✊📄✂️

A simple C++ console-based Rock Paper Scissors game where the user plays against the computer. The user selects Rock, Paper, or Scissors, and the computer randomly generates its choice. The program then compares both choices and displays whether the user wins, loses, or the match ends in a tie.

## Features

* User chooses Rock, Paper, or Scissors
* Computer generates a random choice
* Displays both user and computer selections
* Determines Win, Lose, or Tie
* Simple console-based interface

## Technologies Used

* C++
* VS Code

## How to Run

1. Compile the program:

```bash
g++ rps.cpp -o rps
```

2. Run the executable:

```bash
./rps
```

## Game Rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
* Same choices result in a Tie
